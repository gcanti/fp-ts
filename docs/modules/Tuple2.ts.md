# Bifunctor

## bimap

Map a pair of functions over the two type arguments of the bifunctor.

**Signature**

```ts
export declare const bimap: <E, G, A, B>(mapSnd: (e: E) => G, mapFst: (a: A) => B) => (t: Tuple2<E, A>) => Tuple2<G, B>
```

Added in v3.0.0

## mapSnd

Map a function over the second component of a `Tuple2`.

This is the `mapLeft` operation of the `Bifunctor` instance.

**Signature**

```ts
export declare const mapSnd: <E, G, A>(f: (e: E) => G) => (fea: Tuple2<E, A>) => Tuple2<G, A>
```

Added in v3.0.0
