## IORef (class)

**Signature**

```ts
export declare class IORef<A> {
  constructor(private value: A)
}
```

**Example**

```ts
import { io } from 'fp-ts/IO'
import { newIORef } from 'fp-ts/IORef'

assert.strictEqual(io.chain(newIORef(1), (ref) => io.chain(ref.write(2), () => ref.read))(), 2)
```

Added in v2.0.0
