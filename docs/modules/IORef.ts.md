# IORef (class)

**Signature**

```ts
export class IORef<A> {
  constructor(private value: A) { ... }
  ...
}
```

**Example**

```ts
import { newIORef } from 'fp-ts/lib/IORef'

assert.strictEqual(
  newIORef(1)
    .chain(ref => ref.write(2).chain(() => ref.read))
    .run(),
  2
)
```
