### drawTree

_function_

_since 1.6.0_

_Signature_

```ts
(tree: Tree<string>): string
```

_Description_

Neat 2-dimensional drawing of a tree

_Example_

```ts
const fa = new Tree('a', [
  new Tree('b', []),
  new Tree('c', []),
  new Tree('d', [new Tree('e', []), new Tree('f', [])])
])
console.log(drawTree(fa))
a
├─ b
├─ c
└─ d
   ├─ e
   └─ f
```
